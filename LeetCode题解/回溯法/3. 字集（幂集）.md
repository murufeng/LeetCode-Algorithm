#### 子集/幂集
> 题目：给定一组不含重复元素的整数数组nums，返回该数组所有可能的子集（幂集）。

**说明：解集不能包含重复的子集**。

> 示例:

> 输入: nums = [1,2,3]
> 输出:
> [
>  [3],
>  [1],
>  [2],
>  [1,2,3],
>  [1,3],
>  [2,3],
>  [1,2],
>  []
> ]


- Python代码实现（回溯法）
```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        if not nums:
            return []

        res = []
        def backtrack(indx,tmp_list):
            res.append(tmp_list)
            for i in range(indx,len(nums)):
                backtrack(i+1,tmp_list+[nums[i]])

        backtrack(0,[])
        return res
```
